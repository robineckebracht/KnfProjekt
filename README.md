# KNF Converter (C# / AngouriMath)

Kleines Lernprojekt zur Umformung aussagenlogischer Formeln in
**konjunktive Normalform (KNF)** mittels Äquivalenzumformungen.

## Tech
- .NET
- AngouriMath (symbolische Algebra)

## Ziel
- Parser → Syntaxbaum
- Rewrite-Pipeline (Implies entfernen, NNF, Distributiv)
- Keine SAT-Optimierung, nur logische Äquivalenz
